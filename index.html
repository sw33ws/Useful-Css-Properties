CSS Contain

The contain property allows you to tell the browser that certain aspects of an element shouldn't have an effect anything outside of that element.

Why you might use contain

It is import for things like container queries, but outside of that, can be useful for performance issues as animations within a contained element will not cause layout recalculations on other elements.
